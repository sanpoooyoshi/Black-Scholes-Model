# 日経225 オプション価格シミュレーター (Standalone)

ブラック・ショールズ方程式に基づき、日経225オプションの理論価格算出と損益シミュレーションを行うためのWebアプリケーションです。

## 特徴
- **完全スタンドアロン動作**: JavaScript のみで計算を行うため、PCの電源がオフの状態やオフライン環境でも、スマートフォンのブラウザからいつでも利用可能です。
- **リアルタイム・シミュレーション**: パラメータを変更すると即座に価格と損益グラフが更新されます。
- **モバイル最適化**: スマートフォンでの操作を前提としたレスポンシブデザインを採用しています。

## 使い方

### 1. スマートフォンでの利用（推奨）
GitHub Pages を有効にすることで、以下の URL からいつでもどこでも利用できます。
**URL**: [https://sanpoooyoshi.github.io/Black-Scholes-Model/src/index.html](https://sanpoooyoshi.github.io/Black-Scholes-Model/src/index.html)

### 2. ローカルでの利用
プロジェクトをクローンし、`src/index.html` をブラウザで開くだけで動作します。

## 主な機能
- **理論価格算出**: 現在の日経平均、行使価格、残存日数、ボラティリティ、金利を入力し、コール/プットの理論価格を算出します。
- **期待損益計算**: 「将来の価格」や「将来の残存日数」を設定することで、オプション価格がどう変化するか（期待損益）をシミュレーションできます。
- **損益シミュレーション・グラフ**: 日経平均が変動した際の価格カーブを視覚的に表示します。
