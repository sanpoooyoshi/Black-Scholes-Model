# 日経225 オプション分析ダッシュボード

日経225オプションのブラックショールズ・モデルによる価格計算と、JPX（日本取引所グループ）の公開データに基づいた建玉残高（OI）および機関投資家動向を可視化するダッシュボードです。

## 特徴

- **オプション価格計算**: 原資産価格、行使価格、残存日数、ボラティリティ、金利を入力し、理論価格（Call/Put）を即座に算出。
- **損益シミュレーション**: 価格変動に応じた損益曲線をグラフで可視化。
- **OI 分布分析**: 行使価格ごとの建玉残高（Open Interest）を表示し、市場の支持・抵抗帯を分析。
- **機関投資家動向**: JPXの売買状況からマーケットセンチメントを推測。
- **モバイル対応**: スマートフォンでは計算機能に特化したシンプル表示に自動切り替え。

## セットアップ

### 1. 依存ライブラリのインストール
Python 3.8以上が必要です。
```bash
pip install -r requirements.txt
```

### 2. データフォルダの準備
解析用のデータを以下の構造で配置してください。
- `機関投資家/`: JPXの売買内訳Excelファイルを配置
- `data/oi/`: JPXの建玉残高Excelファイルを配置

### 3. 起動
`start.bat` を実行するか、以下のコマンドを実行してください。
```bash
python main.py
```
起動後、ブラウザで [http://localhost:9001/src/index.html](http://localhost:9001/src/index.html) にアクセスしてください。

## スマートフォンでの利用方法

### 1. PCと同じ Wi-Fi で利用する場合（フル機能）
PCの IP アドレスを使ってアクセスできます。ダッシュボード最上部に表示される接続情報を参照してください。

### 2. 外出先やPCがオフの状態で利用する場合（電卓機能のみ）
GitHub Pages を有効にすることで、以下の URL からいつでも価格計算機能を利用できます。
**URL**: [https://sanpoooyoshi.github.io/Black-Scholes-Model/src/index.html](https://sanpoooyoshi.github.io/Black-Scholes-Model/src/index.html)
※PCがオフの時は、分析データ（OI分布等）の更新は行われませんが、電卓機能は正常に動作します。
